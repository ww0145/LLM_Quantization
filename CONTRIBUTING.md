# Contributing

## Entry Format

Please use the table style used in the topic files:

| **Title & Authors** | **Areas** | **Tags** | **Links** |
| --- | --- | --- | :---: |

Use badges for source/year, areas, and tags. Keep entries concise.

## Rules

- Put each paper in the most relevant root-level topic file.
- Keep one paper to one row.
- Prefer arXiv, conference, project, or official code links.
- Use concise tags such as `PTQ`, `W4A16`, `KV-Cache`, `FP8`, `Outlier-Aware`, `Serving`.
